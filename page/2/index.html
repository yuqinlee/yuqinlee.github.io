Typora 快捷键

1.菜单快捷键

1.1.文件

快捷键 功能
ctrl+N 新建
ctrl+shift+N 新建窗口
ctrl+O 打开文件夹
ctrl+P 快速打开
ctrl+S 保存
ctrl+shift+S 另存为
ctrl+， 偏好设置
ctrl+W 关闭

1.2.编辑

快捷键 功能
ctrl+shift+V 粘贴为纯文本
ctrl+shift+C 复制为纯文本
ctrl+A 全选
ctrl+L 选当前行
ctrl+E 选当前格式文本
ctrl+D 选当前词
ctrl+J 跳转到所选内容

1.3.段落

快捷键 功能
ctrl+= 提升标题级别
ctrl+- 降低标题级别
ctrl+T 插入表格
ctrl+shift+K 代码块
ctrl+shift+D 公式块
ctrl+shift+Q 引用
ctrl+shift+[ 有序列表
ctrl+shift+] 无序列表
ctrl+[/] 列表缩进

1.4.格式

快捷键 功能
ctrl+B/I/U 加粗/斜体/下划线
ctrl+shift+` 代码
alt+shift+5 删除线
ctrl+K 超链接
ctrl+shift+I 插入图片
ctrl+\ 清除样式

1.5.视图

快捷键 功能
ctrl+shift+9 实际大小
ctrl+shift+= 放大
ctrl+shift+- 缩小
ctrl+tab 应用内窗口切换
ctr+/ 源码模式

2.其他快捷键+

快捷键 功能
ctrl+d 选中单词
ctrl+l 选中一行
ctrl+e 选中相同格式的字
ctrl+f 文件内搜索
ctrl+h 替换
ctrl+shift+f 全局搜索

3.自定义快捷键

Windows 路径:C:\Users\[userName]\AppData\Roaming\Typora\conf\conf.user.json
